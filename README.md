# Logistics Shipment Self-Service Portal

## Project Overview
This project is developed for COMP70006 Software Development Practices Assignment 2.  
The selected module is a Customer Shipment Self-Service Portal for a logistics company.

## Main Features
- Customer login
- Shipment tracking using tracking ID
- View shipment status
- Reschedule delivery
- Add special delivery instructions
- View notification messages

## Software Development Practices Used
- Agile Scrum
- Lightweight DevOps
- Automated Testing
- Secure SDLC

## Technology Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL
- Testing: Postman and JUnit
- Version Control: Git/GitHub

## Project Structure
- backend: Spring Boot REST API
- frontend: React web application
- docs: planning documents, diagrams, screenshots, and test cases
